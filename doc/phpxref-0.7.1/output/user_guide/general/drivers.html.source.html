 426  <h1>Using CodeIgniter Drivers<a class="headerlink" href="#using-codeigniter-drivers" title="Permalink to this headline">¶</a></h1>
 427  <p>Drivers are a special type of Library that has a parent class and any
 428  number of potential child classes. Child classes have access to the
 429  parent class, but not their siblings. Drivers provide an elegant syntax
 430  in your <a class="reference internal" href="controllers.html"><span class="doc">controllers</span></a> for libraries that benefit
 431  from or require being broken down into discrete classes.</p>
 432  <p>Drivers are found in the <em>system/libraries/</em> directory, in their own
 433  sub-directory which is identically named to the parent library class.
 434  Also inside that directory is a subdirectory named drivers, which
 435  contains all of the possible child class files.</p>
 436  <p>To use a driver you will initialize it within a controller using the
 437  following initialization method:</p>
 438  <div class="highlight-ci"><div class="highlight"><pre><span></span><span class="nv">$this</span><span class="o">-&gt;</span><span class="na">load</span><span class="o">-&gt;</span><span class="na">driver</span><span class="p">(</span><span class="s1">&#39;class_name&#39;</span><span class="p">);</span>
 439  </pre></div>
 440  </div>
 441  <p>Where class name is the name of the driver class you want to invoke. For
 442  example, to load a driver named “Some_parent” you would do this:</p>
 443  <div class="highlight-ci"><div class="highlight"><pre><span></span><span class="nv">$this</span><span class="o">-&gt;</span><span class="na">load</span><span class="o">-&gt;</span><span class="na">driver</span><span class="p">(</span><span class="s1">&#39;some_parent&#39;</span><span class="p">);</span>
 444  </pre></div>
 445  </div>
 446  <p>Methods of that class can then be invoked with:</p>
 447  <div class="highlight-ci"><div class="highlight"><pre><span></span><span class="nv">$this</span><span class="o">-&gt;</span><span class="na">some_parent</span><span class="o">-&gt;</span><span class="na">some_method</span><span class="p">();</span>
 448  </pre></div>
 449  </div>
 450  <p>The child classes, the drivers themselves, can then be called directly
 451  through the parent class, without initializing them:</p>
 452  <div class="highlight-ci"><div class="highlight"><pre><span></span><span class="nv">$this</span><span class="o">-&gt;</span><span class="na">some_parent</span><span class="o">-&gt;</span><span class="na">child_one</span><span class="o">-&gt;</span><span class="na">some_method</span><span class="p">();</span>
 453  <span class="nv">$this</span><span class="o">-&gt;</span><span class="na">some_parent</span><span class="o">-&gt;</span><span class="na">child_two</span><span class="o">-&gt;</span><span class="na">another_method</span><span class="p">();</span>
 454  </pre></div>
 455  </div>
